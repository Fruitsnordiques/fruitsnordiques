# Projet : fruitsnordiques.com

## Contexte
Site web vitrine / pitch deck interactif de démarrage pour une serre de production horticole
d'un hectare à vocation sociale, éducative et alimentaire.
Technologie : Next.js 14 + Tailwind CSS + Framer Motion + Vercel
Langue : Français (Canada)
Ton : Inspirant, ancré, communautaire, professionnel

## Stack technique
- Framework : Next.js 14 (App Router)
- CSS : Tailwind CSS
- Animations : Framer Motion
- Déploiement : Vercel
- Domaine : fruitsnordiques.com
- CMS : À définir
- Email : À définir

## Agents actifs
- Agent UI/UX + Branding : design système, composants, identité visuelle (`/agent-design`)
- Agent Contenu : rédaction, structure narrative, SEO (`/agent-contenu`)
- Agent Déploiement : CI/CD, DNS, Vercel, GitHub Actions (`/agent-deploy`)
- Agent Accessibilité : WCAG 2.1 AA, performance, Core Web Vitals (`/agent-qa`)

## Règles de travail
- Jamais de template générique Bootstrap ou ThemeForest
- Design system custom basé sur la palette Fruits Nordiques
- Toujours mobile-first
- Commits Git à chaque étape importante
- Français partout (code commenté en français)
- WCAG 2.1 AA minimum
- Core Web Vitals > 90
