# Identité visuelle — Fruits Nordiques

## Positionnement de marque

**Fruits Nordiques** n'est pas une simple serre. C'est un projet de société.

### Territoire de marque
- Nature + Technologie + Communauté
- Résilience nordique + Innovation agricole
- Ancrage local + Vision systémique

### Archétype de marque : Le Gardien / L'Innovateur
- Chaleureux, ancré, fiable, porteur d'avenir
- Langage : direct, inclusif, inspirant, jamais corporatif

## Typographie

- **Titres :** Playfair Display (serif) — évoque la terre, la tradition, la noblesse
- **Corps :** Inter (sans-serif) — lisibilité maximale, modernité discrète
- **Accents :** Montserrat (sans-serif) — labels, boutons, navigation

## Ton éditorial

| Contexte | Ton |
|----------|-----|
| Titres de section | Court, affirmatif, fort |
| Corps de texte | Clair, humain, précis |
| Boutons CTA | Action directe, positif |
| Données et statistiques | Sobre, sourcé, crédible |
| Témoignages | Authentique, premier plan |
