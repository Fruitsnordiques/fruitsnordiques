# Architecture du site — fruitsnordiques.com

## Stack technique
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion (animations)
- TypeScript
- Vercel (déploiement)

## Structure narrative — Page d'accueil (pitch deck interactif)

1. **Hero** — Accrocher en 3 secondes → "Nourrir le Nord. Ensemble."
2. **Chiffre-clé** — Ancrer l'urgence → Stats déserts alimentaires QC
3. **Problématique** — Créer l'empathie → Isolement, insécurité, dépendance
4. **Solution** — Présenter l'approche → La serre comme écosystème (4 vocations)
5. **Technologie** — Crédibilité → Kainon Technologies
6. **Impact** — Convaincre → Économie, écologie, social
7. **Communauté** — Humaniser → Visages, histoires, accès
8. **Formation** — Élargir → Emplois, relève, entrepreneuriat
9. **Roadmap** — Donner confiance → Phases, jalons, vision
10. **CTA** — Convertir → Investir / Partenariat / Contact

## Pages
- `/` — Page d'accueil (séquence narrative complète)
- `/mission` — Vision et mission détaillées
- `/projet` — Description technique du projet
- `/impact` — Données d'impact (économique, écologique, social)
- `/roadmap` — Phases de développement
- `/contact` — Formulaire de contact
